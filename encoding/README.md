# Install

:pushpin: Install [`MP4Box`](https://formulae.brew.sh/formula/gpac)

:bulb: Test if installed

```
% brew info gpac | grep nstalled 
Not installed
```

```
% brew install gpac
```

:pushpin: Install [`ffmpeg`](https://formulae.brew.sh/formula/ffmpeg)

```
% brew install ffmpeg
```
